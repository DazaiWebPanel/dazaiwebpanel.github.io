dropdown header:

To enable Moderation on this channel, just drag any of the sliders to a value greater than 0.
Conversely, dragging all the sliders to 0 effectively disables scanning in that channel. A higher value means you would like the bot to be more strict. WE DO NOT RECOMMEND SETTING ANYTHING HIGHER THAN 15.
We also suggest having the bot be more lenient on the non-toxicity tags as those tend to be less accurate.

Placeholders

Dazai offers many placeholders to simplify the use of message events.(Note Caps matter)
Tip: if you would like to disable a message, replace it with none!
{USERNAME} The user's username
{MENTION} Pinging the user
{ID} The ID of the User
{OLDLVL} The level the user just passed (Only for Levelup Message)
{NEWLVL} The level the user just got to (Only for Levelup Message)
